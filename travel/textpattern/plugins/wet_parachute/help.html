A parachute for the unheeding author

wet_parachute requests the author’s confirmation before it allows the closing of browser windows containing unsaved or modified entries.

Usage:

  1. Textpattern 4.5.3 is required, wet_parachute will not work with any prior version.
  2. Will not work with Opera due to their lack of support for the onbeforeunload event.
  1. Install, enable, enjoy – no configuration required.

    Visit the plugin help page for more detailled instructions.
